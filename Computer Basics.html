Computer Basics

A computer is a programmable machine that can execute a multitude of sequential, arithmetic, and logical operations. These operations are calculated at high speeds while also displaying the result to the user in the desired format.

The figure below represents the basic view of a computer system:

The original purpose for a computer is to calculate data at a very high speed. However, with the rise in new technological advancements modern computers are capable of doing so much more. They have become an integral part of our day to day lives as they are used almost everywhere from schools, businesses, hospitals, and even our cellphones. This is why as technology advances we must make sure we understand how these new pieces of technology work to fully understand how to benefit from them.

    Central Processing Unit

    The central processing unit or CPU is the brain of the computer. The main job of the CPU is to make sure the computer is functioning properly. It does this by processing data fetching and executing information to pieces of the computer. The CPU is comprised of three components the control unit, arithmetic logic unit, and memory.

    CPU Components

  • The control unit handles all processor control signals. It directs all input and output flow, fetches code for instructions and controlling how data moves around the system.

  • Arithmetic logic unit or ALU is that part of the CPU that handles all the calculations such as Addition, Subtraction, Comparisons that the CPU may need. Along with the ALU also performs logical operations, and bit shifting operations.

  • The memory unit is in charge of storing the results that have been calculated by the ALU and storing them back into the memory. A computer has two types of memory primary memory and secondary memory. Primary memory resides within the CPU whereas secondary memory is external to it.

    • Memory

    Computer memory is a physical device that can either store information temporarily, like RAM (random access memory) or permanently, like ROM (read only memory). These memory devices are used by the operating system, software, and hardware of the computer. Memory can be of two type either primary or secondary.

    Volatile vs. non-volatile memory

    Memory can either be volatile or non-volatile memory. Volatile memory such as RAM is memory is also called temporary memory because the information stored in the RAM is lost when the computer or device is powered off. Nonvolatile memory such as ROM is permanent memory because the information stored in ROM is not lost when the computer is powered off.

    Primary Memory

    Definition: Primary memory is the main memory of the computer system. Retrieving data from primary memory is faster than secondary memory because it is the computers internal memory system. However, Primary memory is volatile, and we be lost if not saved before a computer loss of power or shut off.

    Software:

    Software: Software is a set of instructions that tell a computer what to do and how to function. Computer software accounts for programs and procedures of associated with the operations of a computer system. Software provides the instructions to allow hardware devices to function. There is a verity of computer software however, there are two major categories system software, and application software.

    System software

    Definition: The system software acts as an interface between the hardware and the user. This is because hardware and humans communicate in different languages. The software is responsible for converting all human instructions into machine understandable instructions machine language.

  • Operating System
  • Language Processor
  • Device Drivers

    • Application software

    Definition: Application software is computer software that preforms specific task to meet the user. For instance, a MS PowerPoint application is specialized for making presentations. Unlike the system software the application software is not necessary for the computer system to function.

  • Word processing
  • Spreadsheet
  • Presentation
  • Database management
  • Multimedia tools

    • Operating System

    Operating system: The operating system is the first system that is loaded when the computer is powered on and is called the booting process. It manages all the software and hardware functions more specifically basic functions such as storing data in memory, retrieving files from storage devices, scheduling tasks based on priority and more. There are several different operating systems but the most common are windows OS and macOS.

    Language Processor

    Language processor: The language processor is an important function of the system software. It converts all human instructions into machine understandable instructions machine language.

    Types of computer language:

  • Machine level language: The machine level language is also called low-level language is a series of 0’s and 1’s that a computer interpreters as instructions. That series of 1’s and 0’s is called binary code.

  • Assembly Language: The assembly language is a low-level programming language that’s main purpose is to communicate directly with computer hardware. In addition to that assembly languages are designed to be readable for humans.

  • High-Level Language: High Level language are different from low-level languages in that it uses human language as opposed to machine language. Some high-level programming include java, C++, Python, JavaScript and more. This code is than converted into machine code.

    • Types of language interpreters:

  • Assembler: Converts assembly level program into machine level program.
  • Interpreter: Converts high level programs into machine level program line by line.
  • Compiler: Converts high level programs into machine level programs at one go rather than line by line.

    • Device Drivers

    Device Drivers: A device drivers’ main purpose is to control and manage functions for specific devices on the computer. It lets the operating system communicate with specific hardware components. For instance, an external device such as a printer will need a driver for the computer to control its functions.

    Different Types of computers

    Different Types of computers: Computers can vary based on the development process and components. For instance, some computers can be faster and have higher storage capacity and some are slow with smaller storage capacities. When working with computers it is important to know what the classification of the computer.

    Types of Computers

  • Desktop computer:A desktop computer also called a person computer or PC is designed for use by one person in a fixed location. The desktop computer is an average computer that has a smaller storage capacity than more advanced computer such as the supercomputer or mainframe computer. It generally has a CPU, monitor keyboard and mouse.

  • Laptop:A laptop computer is a small PC that is designed to be portable unlike the stationary desktop PC. A laptop has a similar design to any other computer expert that the internal computers are much smaller allowing the shell of the computer to be smaller. The laptop generally has a screen, keyboard, and trackpad that functions the sane way a mouse does.

  • Workstation:A workstation computer is similar in design to the desktop PC in that it is located in a fixed location for use by one person. However, the workstation computer has a larger storage capacity and has a higher performance than a basic desktop PC. It generally has a monitor, mouse and keyboard, improved graphics, and sound cards along with other peripherals.

  • Smartphone/Tablet:As technology continues to advance the ability to create a smaller computer becomes easier. With these advancements comes technology like the smartphone and tablet which are both miniature versions of computer devices.

  • Mainframe computers:Mainframe computers are an even more advanced version of workstation computers because of its very large storage capacity and fast processing speeds. Mainframe computers are very expensive and are often used by banks, railroads, insurance companies, and healthcare industries.

  • Super computers:Super computers are the fastest and most expensive computers on the plant. They have a large capacity and have the fastest processing speed even surpassing the processing power of the mainframe computers.
