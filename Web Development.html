Introduction to HTML

HTML, also known as, Hypertext Markup Language was developed by Tim Berners-Lee. The first version of HTML was released in 1993. Since then HTML has received numerous updates and improvements leading up to the most recent version HTML version 5.0. HTML allows for the user to markup and structure a webpage and its content with the aid of two other languages CSS and JavaScript.

Why use HTML

HTML is a great language to learn as it is the most widely used language in web development. It is simple and easy to learn and understand the basics of the language as there are a lot of resources and guides. HTML is also free to use and is supported by most if not all web browsers.

How to get started?

HTML can be easily accessed through a variety of platforms including Windows, macOS, and Linux. Along with that HTML has a variety of integrated development environments (IDE) to choose from. To find out more about which IDE best suits you visit:Top ides for HTML.

    • Advantages of CSS


    List Advantages of CSS


  • CSS saves time:CSS can be written once and reused again in multiple HTML pages. It also allows the user to define a style for each HTML element and apply it to as many webpages as needed.

  • Easy to maintain: To make a global change, simply change the style, and all elements in all the webpage will be updated automatically.

  • Global web standards: Nowadays HTML attributes are being depreciated and it is being recommended to use CSS. So, it is a good idea to start using CSS in all the HTML pages to make them compatible with future browsers.

  • Platform Independence:CSS can be used on multiple platforms and is supported by most if not all web browsers.

    Introduction to JavaScript

    JavaScript is a lightweight, interpreted, object-oriented programming language. It was developed at Netscape by Brendan Eich in September 1995. JavaScript was developed as a text-based programming language used both on the client-side and server-side that allows that user to make web pages interactive. While HTML and CSS are languages that give structure and style to web pages, JavaScript gives web pages interactive elements that engage a user.

    Why use JavaScript?

    JavaScript is an object-oriented programming language designed to make web development easier and more attractive. In most cases, JavaScript is used to create responsive, interactive elements for web pages, enhancing the user experience.

    How to get started?


    JavaScript can be easily accessed through a variety of platforms including Windows, macOS, and Linux. Along with that JavaScript has a variety of integrated development environments (IDE) to choose from. To find out more about which ide best suits you visit:Top ides for JavaScript.
