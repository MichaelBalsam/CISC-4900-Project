Introduction Microsoft word

Microsoft word was created by Charles Simonyi and Richard Brodie in October of 1983. Charles Simonyi and Richard Brodie were both former programmers that worked on something called Xerox Bravo. Bravo was one of the first “What-You-See-Is-What-You-Get” (WYSIWYG) text editors. Shortly after leaving Xerox and joining Microsoft in 1981 they developed the first version of Microsoft word, version 1.0 which at the time was named Multi-Tool Word. The purpose of Microsoft word is to provide a graphical word processing program that allows users to type and save documents. Along with that the program has built in tools that help the user, such as spelling and grammar checks.

Introduction Microsoft PowerPoint

Microsoft PowerPoint was developed by Robert Gaskins and Dennis Austin for the American computer software company Forethought, Inc. The product originally named “Presenter” was released for Apple Macintosh in 1987 but the product was shortly after named to PowerPoint. A short time later Microsoft acquired the software from Forethought and in 1990 later released an official version of PowerPoint. Microsoft PowerPoint was developed to help the presenter present data and information by using text, diagrams, animation, images, and transitional effects in the form of slides. PowerPoint is used in a variety of professions such as education and business and it is always a good skill to have on a resume.

Introduction Microsoft Excel

Microsoft Excel was developed by the Microsoft Corporation and was released in 1985. Excel is a spreadsheet system, which organizes data in columns and rows that can be manipulated through formulas that allow the software to perform mathematical functions on the data. The main purpose of Microsoft Excel is for data entry and management and is used in a various professions and business operations. Excel is capable of handling and managing large amounts of data and designed to perform basic and complex mathematical computations. Along with that, Excel allows users to create graphical displays to help demonstrate the data on the spreadsheet. Microsoft Excel is a very powerful tool for a variety of business uses and a very good skill to put on a resume.
